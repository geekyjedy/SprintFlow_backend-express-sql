# SprintFlow (Backend)

### Tech Stack :
- ExpressJS 
- MySQL

### Setup :
1. clone the repo
2. run  ```npm i``` to install required modules
3. create db named ```project_management```
4. run  ```nodemon start```

### Environment Variables :

1. ```PORT```
2. ```ACCESS_TOKEN_SECRET```
3. ```REFRESH_TOKEN_SECRET```
4. ```DB_HOST```
5. ```DB_USER```
6. ```DB_PASSWORD```
7. ```DB_NAME```
